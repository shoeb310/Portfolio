Rat in Maze

Rat in a Maze" game is a popular problem that can be solved using backtracking algorithms. The objective of the game is to find a path from the starting point to the destination in a maze, where the rat can only move in four directions (up, down, left, and right) and cannot move through wall. o represent the maze, we can use a 2D array where each cell represents a position in the maze.

Keyword Search system

It reads keywords from a file and constructs a minimal perfect hash table. Then, it processes a text file, searching for these keywords and keeping track of their occurrences. New keywords found in the text are added to the keyword list. Finally, it prints statistics about keywords that appear more than twice in the text.
